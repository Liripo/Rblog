grep使用

, 19 Dec 2019



grep ：文本搜索工具。

常使用参数

-c 显示有多少行匹配到
-v 显示不匹配
-o 只输出文件中匹配到的部分
-i 忽略大小写
-n 显示匹配到的行及行号#很好用，配合sed就可以显示前后几行内容并修改
-A/B <int>:显示后/前int行内容
-e 一次匹配多个内容，如grep -e a -e b
-q 返回0值或非0值

grep递归搜索文件

真的不错

grep "text" ./ -r -n# -r递归搜索

习惯性grep使用正则时才加引号。
